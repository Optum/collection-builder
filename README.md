### Contributing
We welcome any and all kinds of contributions to this project!

#### Development

In the project directory, you can run:

```bash
npm i
```

Installs necessary dependencies.

```bash
npm run storybook
```

Starts up a [Storybook](https://storybook.js.org/) instance on port 6006.
